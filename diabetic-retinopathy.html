AI / Deep Learning Project

AI-Based Diabetic Retinopathy Detection

Built a deep learning medical imaging system to detect diabetic retinopathy from retinal images, supporting both binary classification and multi-class severity staging.

Tools & Technologies

Python TensorFlow ResNet50 EfficientNetB3 DenseNet121 SMOTE Kaggle Dataset

Key Results

95.2%

Binary classification accuracy

0.95

F1-score for DR detection

0.98

AUC-ROC score

95.6%

Multi-class severity accuracy

What I Did

  • Developed CNN models using ResNet50, EfficientNetB3, and DenseNet121.
  • Combined models into a 3-model ensemble to improve robustness and generalization.
  • Handled class imbalance using SMOTE to improve minority class prediction.
  • Used data augmentation, focal loss, fine-tuning, EarlyStopping, and ReduceLROnPlateau.

Impact

Designed with a focus on high recall to reduce false negatives, making the system more suitable for early diabetic retinopathy screening workflows.
