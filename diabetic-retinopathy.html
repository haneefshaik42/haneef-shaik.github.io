AI-Based Diabetic Retinopathy Detection

Developed a deep learning system to detect diabetic retinopathy from retinal images, supporting both binary (DR vs No DR) and multi-class severity classification.

Tools & Technologies

  • Python
  • TensorFlow
  • ResNet50, EfficientNet, DenseNet
  • Kaggle Dataset

Key Results

  • 95.2% accuracy in binary classification
  • F1-score: 0.95 | AUC-ROC: 0.98
  • Improved multi-class accuracy from 91% → 95.6%

Business/Impact Perspective

Designed to assist early screening of diabetic retinopathy by minimizing false negatives, improving clinical reliability and decision support.
